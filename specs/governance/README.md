# Governance Documentation

Community standards, contribution guidelines, and code of conduct.

## Files

- **CODE_OF_CONDUCT.md** - Community standards and expected behavior
- **CONTRIBUTING.md** - How to contribute to OpenTalent

## Purpose

This directory maintains governance documents that guide community participation, ensure respectful collaboration, and establish clear contribution expectations.

## Key Guidelines

- All community members must follow the Code of Conduct
- All contributions should follow the guidelines in CONTRIBUTING.md
- Pull requests require review and approval before merging
- All commits should follow conventional commit format
