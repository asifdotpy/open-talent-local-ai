# User Stories

User-centric requirements and personas.

## Purpose

This directory contains user stories documenting:
- User personas
- User journeys
- Feature narratives
- Acceptance criteria
- User workflows

## Primary Personas

### Candidates
- Interview experience
- Feedback and results
- Community engagement

### Recruiters
- Candidate management
- Interview scheduling
- Assessment reviews
- Hiring analytics

### Interviewers
- Interview monitoring
- Question management
- Real-time assessment
- Report generation

## Stories Format

Each user story follows:
- As a [user type]
- I want to [capability]
- So that [benefit]
- Acceptance criteria: [conditions]
