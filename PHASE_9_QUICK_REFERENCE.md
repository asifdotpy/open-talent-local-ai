## üÜò Quick Troubleshooting

**Ollama won't start?**
```bash
ollama serve
```

**App won't launch?**
```bash
cd desktop-app
npm install
npm run build
npm start
```

**Audio issues?**
- Check microphone permissions
- Test with system settings
- Try different input device
- Restart recording software

**Video quality poor?**
- Use 1920x1080 resolution
- Set bitrate 5000+ kbps
- Check display scaling
- Close other apps
